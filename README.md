# Shoe Budget Manager 👟

A premium, modern desktop application designed to manage your sneaker budget efficiently. Built with Python and CustomTkinter for a sleek dark-themed user experience.

## 🛠 Features
- **Budget Tracking**: Set your total budget and keep it under control.
- **Inventory Management**: Add sneaker models and their prices to your receipt list.
- **Real-time Validation**: Automatically checks if your purchase fits into the remaining budget.
- **Modern GUI**: Clean, archive-style dark interface.
